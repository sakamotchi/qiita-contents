MacにGrailsをインストールします。Javaはインストール済みとします。

Grailsの本家サイトを拝見すると[SDKMANを利用したインストール](https://grails.org/download.html#sdkman)が紹介されていたので、これをやってみます。

まず、Launchpad からターミナルを起動します。
次のコマンドを実行します。

```
curl -s get.sdkman.io | bash
```

次のような出力があって、「Enjoy!!!」と出れば、インストールできていると思います。

![スクリーンショット 2017-09-23 16.30.22.png](https://qiita-image-store.s3.amazonaws.com/0/203704/373abd0e-0958-3cf0-9975-202a4a14a224.png "スクリーンショット 2017-09-23 16.30.22.png")

インストール時の出力に新しいターミナルを起動するか、sourceコマンドを実行すると使えるとあったのですが、新しいターミナルを起動しても使えなかったのでホームディレクトリで次のコマンドを実行しました。

```
source .sdkman/bin/sdkman-init.sh 
```

インストールできたかは、次のコマンドで確認しました。

```
sdk help
```

次のコマンドで最新のGrailsをインストールします。今回はバージョン3.3.1です。

```
sdk install grails
```

インストールできたかは、

```
grails -verson
```
で確認できます。

<追記>
後日、sdkコマンドが実行できなくなっていのたで調べると、どうやら、macは、.bashrcが読み込まれないので、下記の記述を.bash_profileに記載しました。

```
if [ -f ~/.bashrc ] ; then
. ~/.bashrc
fi
```
