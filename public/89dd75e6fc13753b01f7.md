Gebで右クリックを実装した時のサンプルコードです。

```
// 対象要素を取得
def target = $("#target")

// 対象要素を右クリック
interact {
	contextClick(target)
}

waitFor {
	// 右クリックで表示されるコンテキストメニューをウェイト
	$("#contextMenu1").displayed
}

// コンテキストメニューをクリック
$("#contextMenu1").click()

```

試してはいませんが、interactの内には次のページのメソッドも実行できると思います。
